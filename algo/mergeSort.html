Sortowanie przez scalanie

Rekurencyjny algorytm sortowania wykorzystujący metodę dziel i zwyciężaj.

Schemat działania

  1. Podziel zestaw danych na dwie równe części.
  2. Posortuj każdą część przez scalanie, chyba że został tylko jeden element.
  3. Połącz posortowane podciągi w jeden.

Pseudo-kod funkcji:

					
// posortuj Array[start...end]
mergeSort(Array):
	if Array.length > 1:
		center = Array.length / 2
		left = Array[0..center]
		right = Array[center+1.. Array.length-1]
		mergeSort(left)
		mergeSort(right)
		merge(left, right)
					
				
					
// scal left z right
merge(left, right):
	result = empty list;
	while(left.length > 0 and right.length > 0)
		if(left.first <= right.first)
			result.append(left.first)
			left = left[1..]
		else
			result.append(right.first)
			right = right[1..]
	// jedna jest już pusta
	while(left.length > 0)
		result.append(left.first)
		left = left[1..]
	while(right.length > 0)
		result.append(right.first)
		right = right[1..]
	return result
